# AI Engineering Self-Study

A self-directed curriculum for going from zero to production-level AI engineering skill. This is a personal syllabus, study planner, and resource library — not an institutional program.

## How This Repo Is Organized

- Each **Volume** = one semester's worth of material, built sequentially.
- Each **Volume** contains **course folders**. A course folder holds:
  - `<CODE>.md` — the full syllabus (why it matters, prerequisites, weekly plan, project, resources)
  - `Projects/` — the main project spec and any extra project files *(only present if the course has one)*
  - `Assignments/` — practice assignments *(only present if the course has one)*
  - `Quizzes/` — lightweight self-check quiz banks *(only present if the course has one)*

Courses are generated one at a time, in order, and each is fully built (syllabus + weekly breakdown + resources + project) before the next begins.

## Working Through This Repo

1. Read `Program-Overview.md` once to understand the full map.
2. Go volume by volume, course by course, in order — don't skip ahead, prerequisites are real.
3. Each course's `.md` file is self-contained: read "Why This Matters" and "Prerequisites" first, then work through the Weekly Study Plan.
4. Use the Main Project at the end of each course as the actual proof you've internalized the material, not just the weekly mini-exercises.
5. Use the Self-Check section before starting the next course — if you can't honestly say yes to it, revisit rather than push forward.
